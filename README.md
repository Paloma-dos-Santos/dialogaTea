# DialogaTEA

## 📌 Descrição do Projeto
O DialogaTEA é um aplicativo desenvolvido com foco em auxiliar adolescentes com Transtorno do Espectro Autista (TEA) no desenvolvimento de autonomia social e profissional. O sistema propõe simulações de interações sociais, organização de rotinas e apoio à comunicação em ambientes cotidianos e profissionais.

---

## 🎯 Objetivo
Desenvolver uma solução tecnológica inclusiva que contribua para o desenvolvimento da autonomia de adolescentes com TEA, promovendo aprendizado social por meio de interações simuladas e feedback estruturado.

---

## 🧠 Engenharia de Software aplicada
O projeto foi desenvolvido com base em práticas completas de engenharia de software, incluindo:

- Levantamento e análise de requisitos
- Definição de regras de negócio
- Modelagem de sistema com UML
- Histórias de usuário e definição de atores
- Planejamento baseado em práticas ágeis (Scrum)
- Prototipação de interface
